# Dolwood — лендинг

Лендинг для продажи **буковой доски оптом** и **изготовления мебельных заготовок на заказ**.
Стек: **Next.js (App Router) + статический экспорт** — готовый HTML, который индексирует Яндекс и который разворачивается на любом российском хостинге.

## Запуск

```bash
npm install      # один раз
npm run dev      # локально на http://localhost:3000
npm run build    # production-сборка → статика в папке out/
```

Папка `out/` после `npm run build` — это готовый сайт. Заливаете её содержимое на хостинг (Beget / Timeweb / любой статический хостинг или CDN с серверами в РФ).

## Что где менять

| Что | Файл |
|---|---|
| Телефон, Telegram, WhatsApp, email, город | `lib/site.ts` |
| Пункты меню (якоря) | `lib/site.ts` (массив `nav`) |
| Тексты блоков | `components/*.tsx` (Hero, About, Wholesale, Blanks, Advantages, Process, Contacts) |
| Цены на доску | `components/Wholesale.tsx` |
| Палитра (5 цветов) | `app/globals.css` (`:root`) |
| SEO title/description, домен, JSON-LD | `app/layout.tsx` |
| Домен в robots/sitemap | `app/robots.ts`, `app/sitemap.ts` |

## Логотип

Сейчас слева вверху — текст «Dolwood» + иконка-лист. Когда будет файл логотипа:
1. Положите его в `public/logo.svg` (или `.png`).
2. В `components/Header.tsx` и `components/Footer.tsx` замените `<LogoMark/>` + текст на `<img src="/logo.svg" alt="Dolwood" />`.

## Перед публикацией (чек-лист)

- [ ] Заменить плейсхолдер домена `https://dolwood.ru` на реальный — в `app/layout.tsx`, `app/robots.ts`, `app/sitemap.ts`.
- [ ] Добавить **Яндекс.Метрику**: вставьте счётчик в `app/layout.tsx` перед `</body>` (получите код на metrika.yandex.ru). Цели: клик по телефону, открытие мессенджера.
- [ ] Подключить сайт в **Яндекс.Вебмастере** (sitemap.xml уже генерируется).
- [ ] Заменить иконки-заглушки товаров на реальные **фото** доски и заготовок (сильнее всего влияет на конверсию).
- [ ] Зарегистрировать домен на ИП/юрлицо, хостинг — с серверами в РФ.

## Примечание по 152-ФЗ

Сейчас на сайте **нет формы**, собирающей персональные данные, — связь идёт через звонок/мессенджеры (ссылки), поэтому требований по локализации данных не возникает. Если добавите форму заявки — храните данные на российском сервере и добавьте «Политику обработки ПДн» + чекбокс согласия (см. отчёт «Стратегия-вывода-в-онлайн.md»).
